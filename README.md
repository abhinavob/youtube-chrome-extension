# Video Helper Extension

An extension that gives you extended control over video playback speed on any site beyond default limits, shows total duration of YouTube playlists, and speeds up YouTube ads.

## Features

- Extended playback speed control — speed up videos beyond 2x to watch faster (on any site)
- YouTube playlist watch time — see total duration of playlist to estimate watch time
- Auto speed up ads on YouTube — automatically speed up YouTube ads to quickly skip watching them (can toggle on/off)

## Installation

1. Download or clone this repository  
2. Open `chrome://extensions/`  
3. Enable **Developer mode** (top right)  
4. Click **Load unpacked**  
5. Select this project folder  

## Usage

- Go to `chrome://extensions/` > Keyboard shortcuts (on left) to view or set shortcuts for this extension  
- Open any video → control playback speed freely using the keyboard shortcuts  
- Open a YouTube playlist → total watch time is shown  
- Ads will automatically speed up on opening any YouTube video (can be toggled in popup)  

## Notes

- Designed primarily for YouTube, but works on most sites
